--------
This component is designed to properly load environment configuration files and pass it onto the main application kernel so it can be applied to the whole application upon startup. If you are looking to contribute please goto <https://github.com/LaxovePHP/LaxovePHP/tree/master/components/>. This repository is read only and can only be edited by trusted contributors. If a pull request is accepted we will automatically create a pull request to this repository and merge it.
